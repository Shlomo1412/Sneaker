# Building Sneaker

Each version is a separate Fabric mod project. Navigate into the version's subfolder and run the build command.

## 1.20.1

```
cd 1.20.1/Sneaker
gradlew build
```

Output: `build/libs/`

---

## 1.21.10

```
cd 1.21.10/Sneaker
gradlew build
```

Output: `build/libs/`

---

## 1.21.11

```
cd 1.21.11/Sneaker
gradlew build

---

## 1.21.4

```
cd 1.21.4/Sneaker
